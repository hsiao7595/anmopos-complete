# 按摩店 POS 內部報表 MVP

這是第一版內部統計後台，先把 `報表` 資料夾內的照片整理成可查看的月報、年報與照片清單。

## 使用方式

直接用瀏覽器開啟：

```txt
E:\Codex\按摩店POS系統\index.html
```

登入帳號：

```txt
帳號：admin@example.com 或 admin
密碼：admin123456
```

測試用簡短登入也可使用：

```txt
帳號：admin
密碼：admin
```

## 目前功能

- 負責人總覽：兩店進客比較、班別占比、每日趨勢、登記金額
- 櫃台進場：即時登記來賓進場消費，取代紙本來賓進場消費表
- 店長班表：記錄早班 / 中班 / 晚班進客數、上班師傅、空班師傅、流失數
- 消費明細：查詢櫃台登記的所有來賓消費資料
- 報表照片清單
- 月報 CSV 匯出
- 消費明細 CSV 匯出
- 列印報表

## 來賓進場消費表

「櫃台進場」頁面已改為日常 POS 登記表：

- 店別
- 日期
- 進場時間
- 人數
- 服務項目
- 分鐘
- 師傅
- 包廂
- 結束時間
- 金額
- 備註

紙本照片仍保留在「原始照片」頁，可作為舊資料補登參考。
